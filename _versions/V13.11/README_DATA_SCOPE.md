# V13.10 数据范围说明

本版本为 V13.10 审核演示版本，前端页面读取项目根目录 `_data/current` 下的当前共享数据文件。

## 主数据文件

本轮不修改以下五个主数据文件：

- `_data/current/data-core.js`
- `_data/current/data-views.js`
- `_data/current/data-dictionary.js`
- `_data/current/data-audit.js`
- `_data/current/data-nc-workflow.js`

审核包内会提供上述文件的打包副本，并附带打包前后 SHA256 校验结果。

## 展示边界

- 页面金额、项目、科目、活动和 NC 明细均来自当前离线演示数据。
- 本轮不接入真实 NC 接口，不调用真实外部大模型，不新增第三方库。
- AI 助手为规则化本地演示输出，只基于当前页面已计算结果生成经营复核参考。
- 不同角色仅展示其授权范围内项目、明细和助手上下文；单项目角色不会展示授权范围外项目或项目群名称。

## 审核关注点

- 数据 hash 与金额锚点用于证明本轮未改变主数据口径。
- 页面截图、控制台审计、路由审计和权限审计用于证明 V13.10 前端收尾后仍可正常访问。
